# 自动补全功能使用指南

## 功能概述

编辑器现在支持智能自动补全功能，可以帮助您快速输入待办任务的各种格式和内容。

## 触发方式

### 自动触发
- 输入 `- ` 开始创建任务时
- 输入 `@` 添加时间时
- 输入 `T:` 添加用时时
- 输入 `->` 添加依赖时

### 手动触发
- 按 `Ctrl + Space` 手动触发自动补全

## 操作方式

- **TAB键**: 切换到下一个补全项
- **↑↓方向键**: 上下选择补全项
- **Enter键**: 确认选择当前项
- **ESC键**: 取消自动补全

## 补全类型

### 1. 任务模板
- `- [ ] 新任务` - 创建基本待办任务
- `- [x] 已完成任务` - 创建已完成任务
- `- [ ] 任务 @时间` - 带开始时间的任务
- `- [ ] 任务 @时间 T:用时` - 带时间和用时的任务
- `- [ ] 任务 ->依赖` - 带依赖关系的任务
- `- [ ] 完整任务` - 包含所有属性的完整任务

### 2. 时间格式
- 今天/明天的日期
- 标准日期格式 (YYYY-MM-DD)
- 时间格式 (HH:MM)
- 日期时间格式 (YYYY-MM-DD HH:MM)

### 3. 用时格式
- `1h`, `2h`, `4h` - 小时
- `0.5h`, `1.5h` - 小数小时
- `1d`, `2d` - 天
- `1w` - 周
- `1m` - 月

### 4. 常用任务
预设了常见的开发任务模板：
- 需求分析
- 技术调研
- 数据库设计
- API开发
- 前端开发
- 单元测试
- 集成测试
- 部署上线
- 文档编写
- 代码审查

### 5. 依赖任务
输入 `->` 后会显示当前文档中已存在的任务，可以快速选择作为依赖。

## 模板占位符

某些模板包含占位符，选择后会自动选中占位符内容，方便快速修改：
- `${1:任务名称}` - 第一个占位符，默认文本为"任务名称"
- `${2:2023-12-01}` - 第二个占位符，默认文本为日期
- `${3:2h}` - 第三个占位符，默认文本为用时

## 使用示例

1. **创建基本任务**:
   - 输入 `- ` 
   - 选择 `- [ ] 新任务`
   - 继续输入任务名称

2. **创建带时间的任务**:
   - 输入 `- [ ] 开发功能 @`
   - 自动弹出时间格式选项
   - 选择合适的日期格式

3. **添加用时**:
   - 在任务后输入 `T:`
   - 选择合适的用时格式

4. **添加依赖**:
   - 在任务后输入 `->`
   - 从现有任务中选择依赖项

## 快捷键总结

- `Ctrl + Space`: 手动触发自动补全
- `Tab`: 切换下一项
- `↑/↓`: 上下选择
- `Enter`: 确认选择
- `ESC`: 取消补全

这个自动补全功能将大大提高您编写待办任务的效率！
